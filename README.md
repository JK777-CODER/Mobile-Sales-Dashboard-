📱 Mobile Sales Dashboard
📌 Project Overview

The Mobile Sales Dashboard is an interactive Power BI project designed to analyze and visualize mobile sales data. It provides valuable insights into sales performance, revenue trends, top-selling brands, customer purchasing patterns, and regional sales distribution through dynamic reports and visualizations.

The dashboard helps businesses monitor key performance indicators (KPIs), identify growth opportunities, and make data-driven decisions.

🚀 Features
📊 Sales Performance Analysis
💰 Revenue and Profit Tracking
📱 Top-Selling Mobile Brands & Models
🌍 Region-wise Sales Insights
👥 Customer Purchase Trends
📈 Interactive Charts and Visualizations
🎯 KPI Monitoring Dashboard
🔍 Dynamic Filtering and Drill-Down Analysis
